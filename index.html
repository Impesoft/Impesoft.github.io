    Webdesign - stylesheets in CSS3

    Als moderne webmaster kan je er niet omheen: kennis van CSS is absoluut onontbeerlijk! Je kan je website een mooie opmaak meegeven, die je bovendien in één handeling verandert als je dat wenst.
